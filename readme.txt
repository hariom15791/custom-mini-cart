woocommerec custom mini cart 

== Description ==

This is very simple "woocommerec custom mini cart" plugin.

== Installation ==

* Upload the directory '/custom-mini-cart/' to your WP plugins directory and activate from the Dashboard of the main blog.
* When plugin is activated then add short code , where you want to display "MINI Cart"

=========== Step for "HOW CAN USE THIS PLUGIN ?" ===========

Step 1:- Install the "Woocommerece Plugin"

Step 2:- Install the "woocommerec custom mini cart"

Step 3:- Add shortcode or Php code in Header Section 

Shortcode
=====
[custom_mini_cart]

Php
=====
<?php echo do_shortcode('[custom_mini_cart]'); ?>

Step 4:- END
